Support ​

If you run into any issues or want to see any new features, please let us know!

Contributing is as easy as telling us what features or changes you want to see next…

  • GitHub Discussions
  • GitHub Issues
  • Discord Chat
  • General Feedback

GitHub Discussions ​

If you have a question or need help with anything, start a discussion in General or Q&A.

If you want to see a new feature, start a discussion in the Feature Requests section.

GitHub Issues ​

If you run into any problems or find a bug, please open an issue.

These reports are treated with the highest priority.

Discord Chat ​

You can chat with us about anything on discord.

From here you can contact me directly, Shane@111150265075298304.

General Feedback ​

You can submit general feedback via the web form.

This method is not recommended to report issues or request features.

https://cssnr.github.io/feedback/
